# 🚀 VittSetu – MSME Financial Health Card

VittSetu is an AI-powered prototype designed to solve one of the biggest challenges in MSME lending — lack of formal financial history.

Instead of relying on traditional balance sheets, VittSetu uses **alternate data sources like GST, UPI, EPFO, and Account Aggregator (AA)** to generate a **Financial Health Score (0–100)**.

---

## 🌟 Problem

Thousands of MSMEs in India are denied loans because:

* No formal financial statements
* Limited credit history
* Heavy dependency on manual verification

---

## 💡 Solution

VittSetu creates a **document-free, explainable financial profile** for MSMEs using real-time alternate data.

✔ No paperwork
✔ Faster credit decisions
✔ Transparent scoring

---

## 🧠 Key Features

### 📊 Financial Health Score

* Score out of 100 based on 5 dimensions:

  * Revenue Stability
  * Cash Flow Health
  * Compliance
  * Digital Footprint
  * Debt Behavior

### 📈 Animated Dashboard

* Score reveal animation
* Radar chart visualization
* Strengths & risk insights

### 🔮 Scenario Simulator (What-if Analysis)

* Adjust inputs like GST consistency or UPI volume
* See score update in real-time

### 🚨 Risk Alerts

* Smart alerts for declining trends

### ✅ Alternate Data Verified Badge

* Trust indicator using GST + UPI + AA + EPFO

### 🏦 Bank Officer View

* Portfolio dashboard
* Risk filtering
* Explainable loan decision support

### 🤖 Chatbot Assistant (Scripted)

* No backend / AI calls
* Predefined Q&A flows
* Interactive quick replies
* Helps users understand and improve their score

---

## 🛠️ Tech Stack

* Frontend: React.js
* UI: Tailwind CSS / Custom Components
* Charts: Recharts / Chart.js
* State Management: React State
* No backend (Prototype)

---

## 🎯 How It Works

1. MSME enters basic details (GSTIN, PAN, etc.)
2. Provides consent for data access
3. System processes alternate data (mocked)
4. Generates Financial Health Score
5. Provides insights + credit recommendations

---

## 🔐 Data Privacy

* Fully consent-based data usage
* No data stored or shared without permission
* Built considering Account Aggregator framework principles
